Windows 95 Desktop Simulator built with HTML/CSS/JavaScript

Features:
- Draggable windows with proper z-index stacking
- Start Menu with clickable items
- Taskbar showing active windows
- Live clock in the system tray
- Desktop icons (My Computer, Recycle Bin, Notepad, Calculator, About)
- Minimize, Maximize, and Close buttons for each window
- Classic Windows 95 styling with teal and gray colors
- Three separate files: index.html, style.css, script.js
- Clean code with no comments
